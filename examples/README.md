# Тестовые примеры
Здесь находятся файлы с примерами кода на языке Slowrace, а также бенчмарки для них.
- `hello_world.sr` - код Hello World
- `factorial.sr` - код вычисления факториала числа (рекурсивный)
- `factorial_iterative.sr` - код вычисления факториала числа (итеративный)
- `sorting.sr` - код сортировки массива вставками
- `quick_sorting.sr` - код быстрой сортировки массива
- `sieve.sr` - код нахождения всех простых чисел до заданного с использованием решета Эратосфена
- `nbody.sr` - код для решения задачи `n` тел
