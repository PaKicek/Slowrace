# Тестовые примеры и бенчмарки
Здесь находятся файлы с примерами кода на языке Slowrace, а также бенчмарки для них (в файле benchmarks.py).
- `hello_world.sr` - код Hello World
- `factorial.sr` - код вычисления факториала числа (рекурсивный алгоритм)
- `factorial_iterative.sr` - код вычисления факториала числа (итеративный алгоритм)
- `merge_sort.sr` - код сортировки массива вставками
- `quick_sort.sr` - код быстрой сортировки массива
- `sieve.sr` - код нахождения всех простых чисел до заданного с использованием решета Эратосфена
- `fibonacci.sr` - код нахождения всех чисел Фибоначчи до заданного
- `nbody.sr` - код для решения задачи `n` тел
- `jit_const_folding.sr` - код для тестирования эффективности работы jit (метод Constant Folding)
- `jit_arithmetic_identities.sr` - код для тестирования эффективности работы jit (метод Arithmetic Identities)
- `jit_combined_test.sr` - код для тестирования эффективности работы jit (комбинация всех методов)
- `test.sr` - код для локального тестирования


Для запуска бенчмарков необходимо запустить benchmarks.py из этой директории.
