# Slowrace
Реализация компилятора и среды исполнения языка программирования Slowrace, написанная на языке Java. Компилятор преобразует исходный код программ в исполняемый байт-код, запускаемый виртуальной машиной.
## Цели и задачи
Разработать JIT-компилируемый язык программирования с автоматическим управлением памятью и сборщиком мусора, обладающий необходимым функционалом для запуска простых приложений (примеры указаны в каталоге `examples`).

## Функциональные требования
- Поддержка базовых арифметических операций (`+`, `-`, `*`, `/` (целочисленное деление), `%`)
- Условные операторы (`if`, `else`, `elif`)
- Циклы (`for`, `while`)
- Типы данных (`int`, `bool`, `string`)
- Массивы и функции (обозначаются `array` и `func` с типом данных соответственно)
- Поддержка рекурсии (с размером стека до ~20 элементов)
- Поддержка ввода и вывода данных (через аргументы командной строки и через `print` соответсвенно)

Более подробный синтаксис находится в каталоге `docs`.
## Нефункциональные требования
- Запрещено использование LLVM при разработке компилятора
- Виртуальная машина должна поддерживать исполнение программ, в частности тестовых примеров
- Необходимо использовать JIT-компилятор для компиляции программ
- Необходимо использовать Garbage Collector для управления памятью и сборки мусора в виртуальной машине
- Расширение файлов с исходным кодом - `.sr`
- Все компоненты компилятора и среды исполнения программ должны быть написаны на языке Java (в том числе и с использованием любых библиотек)
## Процесс разработки
- Разработка ведётся спринтами (по вторникам и субботам)
- В течение спринтов весь день уделяется исключительно разработке компилятора и среды исполнения языка Slowrace
- Можно работать над языком и в любое другое свободное время, но во время работа над языком обязательна!
